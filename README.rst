Overview
========

New C++ engine for GOLD Parser. It supports unicode and new .egt file format.

Building
========

You can build cppauparser from cmake::

	$ md build
	$ cd build
	$ cmake ..
	$ make install

Alternatively you can use Visual C++ 2010 or newer to build cppauparser using msvc/all.sln.
